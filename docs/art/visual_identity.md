# Chornolis Marches Visual Identity

This folder keeps the first visual direction notes for Chornolis Marches.

The current direction is a dark Ukrainian / Slavic inspired forest frontier: ancient wood, mud, fog, twisted roots, carved border markers, geometric embroidery-like ornament, muted lantern light, deep greens, black shadows, deep red accents and muted gold details.

## Goals

- Keep the project visually grounded in a mysterious forest frontier rather than generic modern fantasy.
- Use Ukrainian geometric ornament as a worldbuilding motif, especially on borders, markers, banners, shields, seals and UI frames.
- Preserve a slightly ominous, ancient, pagan wilderness atmosphere.
- Prefer symbolic, readable compositions that work in Telegram UI.
- Avoid cartoon style, glossy MMO fantasy, modern corporate iconography and over-bright heroic fantasy.

## Main uses

- Telegram welcome screen.
- Telegram bot icon or project emblem.
- README / landing page mood art.
- Future UI frames, banners, cards and section dividers.
