# Signs, Dreams and Quotations

Status: proposed content/design bible for the liminal sign layer.

This document records how Chornolis should use external quotations, familiar motifs and recurring signs without breaking the world voice.

## Core rule

A quotation should become folklore before it reaches the player.

The project can use Lynch/Twin Peaks, Heraclitus, Dante, Shakespeare, map/territory thought and similar sources as structural inspiration. Player-facing text should usually be Ukrainian, local, uncertain and diegetic. The player should feel that villagers, dreams, animals, old places and fires are repeating a sign — not that the game is winking at a media reference.

```text
У Chornolis знання приходить через спостереження, але жодне спостереження не є невинним.
Світ дає знаки, але не гарантує, що знак означає те саме двічі.
```

## Placement decision: the direct owl line

Use the direct line once, and put it at a threshold.

Preferred placement:

- tutorial dream;
- the edge between dream and waking;
- `Край пробудження` or a similar dream exit;
- an inspectable root, pine, bark mark or half-remembered dream sign.

Avoid using the direct line as ordinary starter-camp description. The starter camp should remain the human anchor: fire, watchtower, beginner safety, local supplies and social orientation. It may echo the motif indirectly through rumors and night signs.

### Direct dream-threshold version

```text
На сухій гілці над стежкою сидить сова.

Вона дивиться не на вас — радше крізь вас, у щось за вашою спиною.
Коли ви моргаєте, птаха вже немає.

На корені старої сосни хтось видряпав ножем:

«Сови не ті, ким здаються».

Нижче, дрібнішими літерами:

«І не всі написи лишають люди».
```

### Camp/world echo version

Use this outside the dream when the player is already awake:

```text
Над дорогою сидить сова. Селяни кажуть: у Чорнолісі сови не завжди сови.

Ви чули це як дитячу пересторогу.
Тут, на межі стежки й темряви, вона звучить уже не як казка.
```

The camp echo can appear through:

- NPC rumor;
- old hunter line;
- cat looking upward;
- camp-edge night sign;
- inspectable feather/trace outside the firelight.

Do not repeat the direct quote everywhere. Refrain works through changed echoes.

## Look vs examine rule

`Озирнутися` should orient the player.

`Роздивитися` should add a deeper layer: a practical clue, a richer description, a hidden sign, or a destabilizing detail.

### First observation-feedback seed

Record this for the first observation-learning or tutorial hint moment:

```text
Ви уважно дивитесь.

Здається, ви нічого не зрозуміли.
Але відтепер вам буде важче повірити першому враженню.
```

Use this sparingly. It is a tonal milestone, not a repeated failure message.

## Conditional owl examine copy

Owls already exist as nocturnal ecology pressure and indirect signs. The content layer should build on that, not replace it.

### Conditions

Use special owl examine copy only when at least one of these is true:

- dusk, night or dawn;
- young moon, dark moon, full moon, heavy fog or reduced visibility;
- camp edge, watchtower edge, riverbank edge, old pine, threshold path or other authored liminal place;
- the player is using `Роздивитися`, not only ordinary `Озирнутися`;
- a rate limit or one-time player flag allows the line.

Do not show the strange copy for every owl in every location.

### Generic strange owl examine

```text
Це сова.

Принаймні, так її назвав би той, хто не дивився надто довго.
```

### Darker variant

```text
Сова сидить нерухомо, хоч гілка під нею ворушиться від вітру.

Її очі не блищать у темряві.
Вони чекають, доки блиснете ви.
```

### Camp-safe edge variant

```text
З-за межі вогню озивається сова.

У таборі від цього не стає темніше.
Але край світла здається ближчим.
```

## Fire / magical campfires

Ordinary campfires should mostly stay practical: light, warmth, rest, fuel, fading, ash.

Old seeded campfires and magical campfires may carry a second layer. This belongs mostly to `Роздивитися`, not ordinary `Озирнутися`.

### Conditions

- magical campfire, old seeded campfire, ritual fire or authored omen fire;
- dusk/night/dawn or a special dream/lucid context;
- one-time or low-rate reveal per feature/player;
- no ordinary player-made debug campfire gets the strange line by default.

### Fire lines

```text
Не кожен вогонь гріє.
```

```text
Багаття показує не лише тих, хто сидить біля нього.
```

```text
Вогонь іде поруч, доки ви не озирнетесь.
```

### Look/examine split

`Озирнутися`:

```text
Вогнище горить рівно, майже без тріску.
```

`Роздивитися`:

```text
Полум'я тримається без хмизу.
Воно дає світло, але тіні навколо нього не поспішають відступати.
```

Rare magical layer:

```text
Багаття показує не лише тих, хто сидить біля нього.
На мить здається, що за межами світла хтось теж гріє руки.
```

## Fishing and deep water

Fishing can carry the Lynchian “ideas/fish/depth” motif without naming it.

Use only near deep water, dusk/dawn/night, repeated silence, unusual catches or authored fishing spots.

```text
Мілка вода дає дрібну рибу. Глибока — те, що не завжди варто витягати.
```

```text
Старий рибалка каже: якщо довго сидіти тихо, річка починає думати за тебе.
```

```text
Не все, що клює, є рибою.
```

## Structural motif bank

### Heraclitus / hidden nature

```text
Ліс не бреше. Він просто не пояснює.
```

```text
Те, що сховане, тримає світ міцніше за видиме.
```

```text
Знак не відповідає. Знак лише показує, куди дивитись.
```

### Dante / dark forest and lost path

```text
Ви не пам'ятаєте, де саме збилися зі стежки.
```

```text
Пряма дорога лишилась десь позаду.
Тут усі стежки здаються бічними.
```

```text
Ліс почався раніше, ніж дерева.
```

### Shakespeare / reversed signs

```text
Не все чисте є добрим. Не все темне є ворожим.
```

```text
У Чорнолісі гарні знаки часто ведуть не туди.
```

```text
Те, що пахне димом, може бути домом. А може — пасткою.
```

### Map/territory

```text
Карта не знає, що ліс змінився.
```

```text
Те, що намальоване стежкою, може бути лише чиєюсь надією.
```

```text
Мапа показує місця. Чорноліс показує, де ви.
```

### Ukrainian mythic layer

Viy / dangerous sight:

```text
Не кожне око варто відкривати.
```

```text
Деякі речі сліпі лише з милости.
```

```text
Воно не бачило вас, доки ви не захотіли побачити його.
```

Mavka / wrong presence:

```text
Вона мала голос людини, але тінь — ні.
```

```text
На воді відбивався ліс. Її там не було.
```

```text
Не кожен, хто кличе на ім'я, пам'ятає ваше ім'я.
```

Chugayster / strange protector:

```text
Уночі хтось танцював між деревами.
Вранці навколо багаття не лишилося жодного сліду.
```

```text
Старий біля вогню сміється так, ніби тріскаються сухі гілки.
```

Use as small local speech, not exposition dumps.

```text
Сови тут розумні. Або щось розумне носить совині обличчя.
```

```text
Не ходи за птахом, який не лишає тіні.
```

```text
Якщо карта й ліс сперечаються — вір ногам. Але не завжди своїм.
```

```text
Не дивись довго в те, що вдає, ніби не бачить тебе.
```

```text
Біля старої сосни всі чують різне. Це нормально. Гірше, коли всі чують одне.
```

```text
Старі кажуть: коли сова мовчить надто довго, це вже не сова слухає.
```


```text
Рибалки не люблять глибокої тиші. Кажуть, вона клює першою.
```

## Implementation hooks

Prefer a small authored content layer before broad systems.

Suggested content keys:

```text
liminal.owl.direct_threshold_quote
liminal.owl.world_echo
liminal.owl.examine_generic_strange
liminal.owl.examine_dark_variant
liminal.owl.camp_edge_sign
liminal.observation.first_uncertain_feedback
liminal.fire.magic_examine_layer
liminal.fire.old_campfire_omen
liminal.fishing.deep_water_warning
liminal.map.territory_warning
```

Suggested conditions:

```ts
type LiminalSignCondition = {
  dayparts?: Array<'DAWN' | 'DAY' | 'DUSK' | 'NIGHT'>;
  moon?: 'dark' | 'young' | 'full' | 'any';
  weatherTags?: string[];
  featureTags?: string[];
  locationTags?: string[];
  requiresExamine?: boolean;
  oncePerCharacter?: boolean;
  cooldownHours?: number;
};
```

Avoid a migration for the first pass if existing feature data, tutorial flags or world event history can carry the needed one-time/cooldown behavior.

## Anti-patterns

Do not:

- repeat the direct owl quote in many locations;
- explain what owls “really are” in a lore entry;
- make every ordinary owl magical;
- make every ordinary campfire uncanny;
- turn fishing into constant wisdom text;
- add modern author names to in-world UI;
- replace Ukrainian mythic texture with reference collage.
