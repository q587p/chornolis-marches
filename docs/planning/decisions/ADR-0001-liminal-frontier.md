# ADR-0001: Chornolis Marches is a liminal frontier

## Status

Accepted

## Context

The name “Порубіжжя Чорнолісу” implies a threshold, not just geography.

## Decision

Use liminality as a core design principle.

## Consequences

- Night/day matters.
- Settlement/wilderness boundary matters.
- Human/non-human boundary matters.
- Information should often be partial.
- Mythic encounters should feel like crossing thresholds.
