## Active Lane Shape

`0.16.x` starts NPC mentorship from the completed `0.15.x` attention and learning foundation.

Mentorship is teacher/student attention and learning context. It is not `TravelGroup`, not automatic group movement, not hidden-route sharing, not a public skill sheet and not a broad profession economy.
