# MAP-WILLOW-002: Willow under-bridge and watchtower integration

## Goal

Review whether the first `Вербові Низи` / Willow Floodplain pocket should connect more clearly to the authored under-bridge place and the starter-camp watchtower view, without opening the settlement gate or adding a new system.

This is a follow-up planning slice after the first Willow contour and landmark/resource pass. It should make the map easier to read and remember, not expand into fishing, observation progression, economy, combat, theft or settlement access.

## Possible Slice

- Add or adjust watchtower view text so players can recognize that the wet willow pocket exists near the old bridge and riverbank.
- Review under-bridge copy and exits so it stays a distinct lower place, not an accidental shortcut through the Willow pocket.
- Consider a small inspectable hint, sign, or visible riverbank detail that points toward the floodplain route after players discover the bridge area.
- Keep any new connection one-way-informational unless a separate route slice deliberately adds movement.

## Out Of Scope

- No settlement gate bypass.
- No fishing mechanics.
- No observation or learning progression.
- No economy, theft, combat, new visibility system or new creature loop.
- No casual `z` changes beyond existing authored vertical/lower locations.

## Acceptance

- The under-bridge location remains separate and intentionally reachable only through its current authored exits unless a later route task changes that explicitly.
- The starter watchtower or nearby riverbank prose gives a clearer mental map of bridge / river / Willow Floodplain relationships.
- `docs/world/world_map.md` remains accurate if any map data changes.
- Seed, route and HTML-content checks pass for any world-data edits.
