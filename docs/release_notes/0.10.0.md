# 0.10.0 — Rabbit reproduction and overgrazing

## Summary

This update starts the first real ECO-001 loop: rabbits are now part of the living ecology, not only static low-level targets.

## Gameplay / UX

- Adult rabbits can reproduce during world ticks when local food exists.
- Predator pressure, local danger and crowding reduce uncontrolled growth.
- Too many rabbits in one location consume edible resources in the background.
- World tick reports now include rabbit births and overgrazing counters.

## Technical

- Added a rabbit reproduction and overgrazing pass to `worldTick`.
- Added starter rabbits to regular seed data creation, not only `/reset`.
- Reset starter rabbits now begin as mature adults with sex assigned.
- Kept the first implementation data-driven through existing creature and resource records; no schema migration was needed.

## Planning

- ECO-001 moved from `next` to `testing`.
- Follow-up work remains for tuning, migration/spread, explicit recent hunting pressure and visible location/region degradation.
