# 0.10.1 — Rabbit litter tuning

## Summary

This patch makes rabbit reproduction more aggressive and closer to the real-world overpopulation pressure we want to test.

## Gameplay / UX

- A successful rabbit breeding event now creates 5-10 offspring instead of 1-2.
- Predator pressure, local danger and local crowding still constrain runaway growth.

## Technical

- Added `WORLD_RABBIT_MIN_LITTER_SIZE`.
- Added `WORLD_RABBIT_MAX_LITTER_SIZE`.
- Kept the default litter range at 5-10.

## Planning

- ECO-001 and ecology docs now record the new litter size.
- Icebox now tracks future opportunistic rabbit feeding on small birds or carrion once birds, carcasses and deeper attack behavior exist.
