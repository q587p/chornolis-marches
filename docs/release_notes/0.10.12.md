# 0.10.12 — Seed typecheck and deploy announcements

Technical patch for the predator ecology release train.

## Changed

- Starter animal definitions now live in one shared module used by both `npm run seed` and `/reset`.
- Deploy announcements no longer send an older version notification if a same-or-newer `DEPLOY:x.y.z` event already exists.
- Render deployment docs now include a checklist for when to run migrations, when to run seed and the manual deploy order.
- The gatherable `herbs` resource is now displayed as `лікарські трави`, so players do not confuse it with ecology-only `grass`.

## Fixed

- `npm run seed` no longer has local starter-animal declarations conflicting with shared imports.
- `npm test` now type-checks `prisma/seed.ts` without executing the seed, so seed-only TypeScript errors are caught before deploy.
