# 0.10.2 — Paginated long Telegram outputs

## Summary

This patch makes long Telegram debug and news outputs navigable instead of splitting them into a pile of separate messages.

## Gameplay / UX

- `/all` and `/all dead` now show paginated results with `Далі` / `Назад`.
- `/locationAll` now pages through the full location list.
- `/news` now shows the latest entry in full and adds an archive of version buttons.
- The news archive can page deeper into history and open the full text of a selected version.

## Technical

- Page builders target about 3300 characters, leaving room below Telegram's 4096-character message limit.
- Pagination callbacks reread current data and edit the existing message where possible.
