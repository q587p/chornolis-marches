# 0.10.3 — Rabbit pacing and spread

## Summary

This patch slows the rabbit lifecycle from a seconds-scale lab burst into a more readable fast simulation, while making overpopulation spread across the map more reliably.

## Gameplay / UX

- Rabbit reproduction is checked every 40 world ticks by default.
- Rabbit offspring now become adults after 240 world ticks by default.
- Adult rabbits live much longer before old age becomes dangerous.
- Overcrowded rabbit locations can spread excess non-child rabbits into neighboring cells.
- World tick reports include `rabbitsSpread`.

## Technical

- Added `WORLD_RABBIT_SPREAD_EVERY_TICKS`.
- Added `WORLD_RABBIT_MAX_SPREAD_PER_LOCATION`.
- Rabbit spread cancels active queued/running creature actions before relocating the rabbit.

## Planning

- ECO-001 and ecology docs now describe rabbit timing and spread behavior.
