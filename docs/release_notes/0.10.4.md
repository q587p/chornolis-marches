# 0.10.4 — Ecology web stats

## Summary

This patch adds a web statistics page for watching the ecology simulation while tuning ECO-001.

## Gameplay / UX

- `/stat` shows ecology totals, animal age groups, resources, recent births, spread, overgrazing and deaths.
- The page auto-refreshes roughly once per minute.
- The page includes a manual refresh link.
- The main status page now links to `/stat`.
- Telegram `/stat`, `/stats` and the menu stats button show a short ecology summary and a button to open the full web page.
- Animal corpses now remain visible longer for ecology observation and future freshening/scavenger loops.
- Rabbit reproduction no longer has a global population hard cap; local crowding still lowers birth chance and drives spread.

## Technical

- Added `src/services/ecologyStats.ts`.
- Added `/stat.json` for raw ecology statistics.
- `/stat` estimates recent rates from recent `World Tick` events.
- Added `PUBLIC_BASE_URL` for configurable public links in Telegram messages.
