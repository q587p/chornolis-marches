# 0.10.5 — World map v7 and starter ecology

## Summary

This patch applies the v7 world map update and makes the initial rabbit population more useful for ecology observation.

## World

- Added the split `prisma/data/world/*.json` seed data as the active world seed source.
- Added several riverbank / under-bridge map changes, including `under_bridge_18_05` at `z = -1`.
- Removed direct vertical bridge-deck links to the under-bridge location; under-bridge travel now goes through nearby riverbank locations.
- Added Ведана as a female herbalist-like NPC while keeping the current Здравомир and Дід лісовик setup.
- Added the ASCII map rendering script and map editing notes.
- Added first seed tests: `npm test` validates split world seed references and `npm run test:db` checks that the current database has the configured start location.

## Ecology

- Starter rabbits now include children, young rabbits, adults, old rabbits and a couple of corpses.
- `/resetWorld` recreates the same mixed starter rabbit population for easier observation.

## Technical

- `getStartLocationId()` now reads `meta.startLocationKey` from split seed metadata when available.
- Unique creature seed/reset data can now carry `sex`.
- Action, recovery, world-tick and debug/reset writes now tolerate already-removed records instead of failing the whole loop with Prisma `P2025`.
- Lisovyk wake/sleep regional notifications and world events now use in-world growled speech about depleted and recovered resources.
- CI runs the static world seed test before build.
