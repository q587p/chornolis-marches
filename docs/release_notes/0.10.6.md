# 0.10.6 — Unknown command fallback

## Summary

This patch starts 0.10.6 with a small Telegram UX improvement.

## Telegram UX

- Unknown slash commands now get a clear reply instead of silence.
- The reply points players toward `/help` and `/menu`.
- `/respawn` now has a specific placeholder message explaining that the actual respawn feature is still planned.

## Ecology

- Added mixed starter mice to seed/reset: children, young mice, adults, old mice and corpses.
- Slowed mice from the old ultra-fast lifecycle into a readable short lifecycle: still much faster than rabbits, but no longer instant.
- Mice can now reproduce and spread when overcrowded.
- World tick reports and ecology stats now include `mouseBirths` and `miceSpread`.

## Technical

- Added `src/handlers/fallback.ts` and registered it after all normal handlers.
- Seed upserts now use bounded concurrency through `SEED_CONCURRENCY`, defaulting to 12.
- Starter animal creation now uses `createMany` batches instead of one insert per creature.
