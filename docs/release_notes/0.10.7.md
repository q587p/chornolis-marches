# 0.10.7 — Safer reset and tick commands

## Summary

This patch adds a safety confirmation for reset, restores runtime tick debug commands, and retunes the first ECO-001 herbivore loop.

## Admin UX

- `/reset` now asks for inline confirmation before it actually resets the world.
- Only the user who requested reset can press that confirmation button.
- Cancel leaves the world untouched.

## Fixed

- `/tickGet` and `/tickSet` work again.
- The unknown-command fallback is now registered after tick command handlers, so it no longer catches valid tick commands.
- Changelog headings use Holocene Calendar dates again.

## Ecology

- Rabbits now reproduce less often by default: every 120 world ticks, with 5-10 offspring per litter.
- Mice reproduce faster than rabbits: every 20 world ticks, with 5-10 offspring per litter and earlier maturity.
- Multiple adult females in one location can produce multiple litters if a mate is present.
- Added `grass` as an ecology-only forage resource. It feeds herbivores but does not appear in player gather menus.
- Bridge spans have no grass; the under-bridge riverbank still can.
- Severe local depletion creates a visible `Винищена трава` feature and slows recovery.
- `meadow_16_05` starts exhausted so the effect is immediately visible.

## Technical

- Bumped package metadata to `0.10.7`.
