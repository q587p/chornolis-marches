# 0.11.10 — Ecology pressure, track details and admin polish

Release date: 12026-05-27

## Player-facing changes

- Hungry animals now matter more: herbivores search for food or move away more actively, predators hunt more urgently, and animals that stay above the starvation threshold can die of hunger.
- Starvation leaves a corpse, cancels that creature's pending actions and writes an `Animal starved` world event.
- `/stat`, web `/stat` and `/stat.json` now include starvation death counters.
- `/track` is again a quick scan: it shows who went where, without age details.
- `Роздивитися` a location only says that tracks are present. The follow-up `Роздивитися` for tracks, `/examine tracks` or `роздивитися сліди` shows approximate track age.
- `/who` now has pagination in Telegram when the active-character list grows.
- Web `/who?page=0` and `/who.json?page=0` expose the same paginated active-character list.
- NPC/herbalist social auto-signals no longer target themselves. They can still make targetless location gestures such as `Здравомир киває.` or ask for quiet.
- Targetless social-signal events no longer show the internal `location` marker in `/chat`, web `/chat` or `/chat.json`.

## Scribe/admin changes

- Pending scribe prompts such as name-rejection comments and requested teleport destinations can be cancelled with `/cancel`, `/skasuvaty`, `/vidminyty`, `cancel`, `skasuvaty`, `vidminyty`, `скасувати`, `відмінити`, `відміна`, `стоп` or `не треба`.
- Admin auto-toggle world events now name both the scribe and the affected character in the title, with player/scribe ids in the event description.
- Name approval/rejection and admin auto-toggle messages use gender-aware past-tense verbs for the scribe, including plural/they forms.
