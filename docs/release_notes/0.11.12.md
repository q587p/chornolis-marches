## Design notes

- Effects are intentionally modest. This makes gathered resources useful now without replacing rest, future medicine, cooking, crafting or herbalism.
- Exact values remain implementation details; player-facing text describes the result diegetically.
- Dropped resources use the current location resource model for now. Later item instances should preserve origin, quality, timers and richer ground-object behavior.
