# 0.11.4 — Social signals, chat menu and planning refresh

Release date: 12026-05-26

## Gameplay / UI

- Added MVP **Сигнали** for target-focused social gestures:
  - Усміхнутися
  - Засміятися
  - Кивнути
  - Вклонитися
  - Вказати
  - Насупитися
  - Зітхнути
  - Помахати
- Target action cards now show contextual quick signals and a **Ще сигнали** menu.
- Social signals send separate messages to the actor, the target player and other observers in the same location.
- Added `💬 Репліки` to the menu keyboard for opening the existing `/chat` view.

## Admin / Docs

- Deduplicated `/adminHelp` so there is one canonical admin command list.
- Added `/stat`, `/chat` and `/addCampfire` to admin help and admin-command documentation.
- Added `docs/systems/social-signals.md`.
- Updated `docs/planning/next.md` to show the active next lane instead of only one older top item.
- Added backlog notes for `/reply`, quick replies and custom addressed replies.

- `npm run build`
