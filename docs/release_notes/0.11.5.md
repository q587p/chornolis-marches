# 0.11.5 — Ukrainian command aliases and button parity

Release date: 12026-05-26

## Gameplay / UI

- Added Ukrainian text and slash aliases for common actions:
  - `озирнутися`, `роздивитися`, `де я`, `хто я`;
  - `статистика`, `репліки`, `час`, `новини`, `допомога`;
  - `північ`, `південь`, `схід`, `захід` and movement phrases such as `йти на південь`;
  - `збирати трави`, `збирати ягоди`, `збирати гриби`;
  - `відпочити`, `черга`, `скасувати`, `очистити чергу`;
  - `сказати ...`, `увімкнути авто`, `зупинити авто`.
- Added MUD-style target commands for visible nearby targets, including examples such as `кивнути 1`, `помахати мандрівник`, `атакувати вовка`, `роздивитися труп`, `освіжити труп` and `підібрати труп`.
- Existing English slash commands and Telegram buttons remain available.

## Consistency

- Menu aliases reuse canonical handlers/renderers:
  - `Статистика` -> `/stat`;
  - `Репліки` -> `/chat`;
  - `Час` -> `/time`;
  - `Новини` -> `/news`.
- Movement and gathering aliases reuse the same command flow as `/north` / `/south` / `/gather`.
- Unknown text does not replace the existing fallback behavior.

- Updated planning notes for command/button parity and a future shared command/action registry.
