# 0.11.6 — Fire, light, nearby text and scribe access

Release date: 12026-05-26

## Gameplay / UI

- `/addCampfire` now creates a separate timed debug campfire on each call instead of reusing one feature key.
- Timed debug campfires burn for 8 in-game hours / 16 real minutes and warn during their final 2 in-game hours / 4 real minutes.
- Torches are world items rather than guaranteed player inventory; near a campfire, a carried torch can be lit or refreshed.
- Lit torches give local light for 5 in-game hours / 10 real minutes and warn during their final in-game hour / 2 real minutes.
- Campfire light and lit torches both count as local light for revealing nearby targets in the current location view.
- Seed data includes several loose torches in forest, dry luka and riverbank locations; they appear under `Лежить` and can be picked up without a gather chance roll or action delay when the character is not exhausted.
- Seed data includes згаслі campfires in the forest and dry luka.
- Expired timed campfires remain as `Згасле вогнище`, stop giving light/rest benefits, and can be relit with a burning torch.
- `/start`, `/reset` and ordinary bot startup no longer restore torches directly into player inventory; torches can be found in the world or added by the scribe/admin `/addTorch` command.
- `/addTorch [player]` and `/addTwigs [player]` are listed in `/adminHelp` and add a torch or twigs to the current or named player's inventory for configured scribe/admin Telegram IDs.
- `/addCampfire [locationKey|x,y,z|player]` can create a campfire away from the caller's current location; without a parameter it still uses the current location.
- `Додати хмиз` and `/add twigs campfire` are reserved as placeholders for the next firewood implementation.
- Ordinary nearby output avoids technical `тварина/об’єкт` labels, keeps living beings in `Поруч`, and separates corpses into `Лежить`.
- Target lists and target buttons show the active or last known creature action where available.
- Unknown ordinary text now receives a short fallback reply with `/help` / `/menu` hints and close alias suggestions.
- Detailed `/examine` now hints when tracks are present and shows a `🔎 Сліди` action; `/track`, `/examine tracks` and matching text aliases perform the time/stamina-spending track search.
- Duplicate target buttons no longer add numbering in the ordinary player view.
- Detailed `/examine` movement output keeps `Рух поруч`, but now uses Ukrainian age labels:
  - `дитинча`;
  - `молодняк`;
  - `доросла особина`;
  - `стара особина`.
- Creature movement/action text is normalized to third person, such as `йде на схід`, `збирає лікарські трави`, `їсть`, `відпочиває`, instead of internal first-person queue text like `йдемо на схід`.
- The main reply keyboard now places `☰ Меню` where `Персонаж` used to be.
- Character details remain available through `/me`, Ukrainian/text aliases and the `❤️`/`⚡` status button.
- Corpse inventory labels now include sex-aware Ukrainian names for foxes and wolves: `труп лиса`, `труп лисиці`, `труп вовка`, `труп вовчиці`.
- Admin/debug commands now require scribe access. Configured `ADMIN_TELEGRAM_IDS` still work, and the hidden `/adminSet <secret>` command can grant the current character the `Писар Порубіжжя` role when `ADMIN_SET_SECRET` is present in the environment.
- `/adminHelp` is hidden from the global Telegram command menu, then restored as the valid lowercase `/adminhelp` chat-scoped command for chats whose player has `Писар Порубіжжя` access. The server still accepts both `/adminHelp` and `/adminhelp`.
