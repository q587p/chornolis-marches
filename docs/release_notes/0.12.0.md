# 0.12.0 - Dream tutorial - 12026-05-27

## Summary

This release adds the first playable newcomer tutorial slice as a separate dream map layer.

## Added

- Added `Дрімотна Межа`, a tutorial dream region on `z = -13`.
- Added first-run tutorial entry after character onboarding.
- Added `/sleep tutorial` to enter or return to the saved tutorial position.
- Added `/sleep` routing to the tutorial while the tutorial is incomplete.
- Added `/wake` / `Прокинутися` to leave the dream and restore a valid real-world location.
- Added reusable visible locked exits with blocked movement reasons and queued-move lock checks.
- Added `Брама Сну`, an interactive gate opened with `/open` / `Відкрити` for about 30 seconds.

## Changed

- The closed settlement gate now exposes a visible locked east exit for the same locked-exit behavior.
- Updated onboarding, input-alias and map documentation around the tutorial flow.
- Removed stale top-level split seed JSON duplicates from `prisma/data/`; active world seed data now lives unambiguously under `prisma/data/world/`.

## Notes

The tutorial branches for social signals, tracking/combat observation, gathering, fire/light and rest remain future work.
