# 0.12.1 - Tutorial dream polish and visible actions - 12026-05-27

## Summary

This patch polishes the first tutorial dream flow, keeps early controls focused, makes dream gate state changes more visible, and records immediate pickup/drop actions more clearly.

## Changed

- The first tutorial dream location now shows only the location overview action and the south movement button.
- Inventory, examine, help, menu and status buttons are hidden from that first tutorial step.
- The first tutorial dream location text now points players toward the only visible south path.
- First tutorial spirit voice comments now react to forward and backward movement in the dream and are logged as NPC speech events.
- The tutorial dream gate is now named `Брама Сну` instead of `Сонні ворота` to avoid the unintended "sleepy gate" reading.
- The visible wake-up button is no longer offered during the early gate step; direct wake commands remain available for manual exits.
- Direction reply-keyboard buttons now wrap visible locked direction labels in compact parentheses while the lock is active.
- The tutorial hub now has a feature-level wake-up prompt, with voice text explaining that players can return to the saved dream position later.
- Expired tutorial dream gates now close through an explicit world event and local dream-location notification.
- Track-detail buttons now use a clear `Сліди` label, and tutorial dream voices react when a player studies tracks closely.
- The second tutorial step now explicitly points players to the south button or text command.
- Immediate pickup and drop item actions now notify other players in the location, write world events, and appear in admin recent-action history.
- Lisovyk depletion warnings now ignore artificial torch/twigs resources and use more natural Ukrainian resource wording.
- Public `/stat` character rows no longer expose whether a listed character is player-controlled or NPC-backed.
- The character card now exposes a Sleep button, and unavailable normal sleep offers a direct tutorial sleep button.
