# 0.12.10 - Beginner tutorial return prompts - 12026-05-28

This patch improves beginner guidance around returning to the tutorial dream.

## Changed

- `/start` now reminds unfinished characters that `/sleep tutorial` returns them to the tutorial.
- `/help` now adds a direct tutorial button for characters who have not completed the tutorial yet.
- Unknown-input fallback messages now include a tutorial-return hint for unfinished characters.
