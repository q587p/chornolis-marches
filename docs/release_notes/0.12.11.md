# 0.12.11 - Prepared character names - 12026-05-28

## Added

- New character onboarding now lets a player choose a prepared character name or enter a custom name after pronoun selection.
- Added a first curated prepared-name pool with stored Ukrainian case forms, origin and rarity.
- Prepared names are hidden once used and are saved as pre-approved by scribes.
- Custom name entry now shows a warning before free text and rejects exact duplicates plus a first set of creature, spirit, sacred and famous-name conflicts.
- Added a small regression test for prepared-name availability and forbidden custom names.

## Notes

- This is the first playable slice of the name registry plan. The prepared-name pool is still code-level content; later work can move it into data or database-backed records with reservations, regional layers and richer scribe review.
