# 0.12.9 - Border marker ecology report - 12026-05-28

This patch adds the first public in-world ecology report while keeping full service statistics behind scribe/admin access.

## Added

- Inspectable border markers now show a limited wildlife report with rough current animal presence, corpse presence and recent birth/death notes.
- Added `/examine sign`, `/examine border marker` and Ukrainian aliases for the same marker inspection flow.
- Added near-term planning for a local console client that can later run command smoke tests before a full MUD server exists.
- Added parser regression tests for shared text aliases so command behavior can be checked without Telegram.

## Changed

- Exact border-marker ecology values are only shown to scribes/admins with technical details enabled.
