# 0.13.1 - Forbidden-name normalization - 12026-05-29

## Changed

- Custom-name validation now canonicalizes obvious separator tricks before checking forbidden names: casing, apostrophe-like characters, internal whitespace and hyphen splitting are normalized for the blocked-name match.
- Expanded the first forbidden-name examples for creature, spirit, sacred, famous-character and common-word names.
- Updated character-name and onboarding documentation for the stricter custom-name guardrail.
- Simplified `/adminHelp` by removing ordinary player-help commands and grouping scribe/admin tools by purpose.
- The first tutorial dream location now uses a focused starter keyboard with only `Look` and the visible south movement button.
- Added `пів` as a Ukrainian text alias for south movement and included direction aliases in unknown-command suggestions.

## Tests

- Added regression cases for spaced-out or separator-split forbidden names.
- Added a guard that all prepared character names remain valid custom-name inputs.

## Notes

- This is still exact blocked-name normalization, not broad fuzzy matching.
