# 0.13.10 - NPC hunter route plan foundation - 12026-05-30

This patch adds the route-planning slice for the future NPC hunter loop.

## Added

- Added `src/services/npcHunter.ts` with a hunter route-plan helper.
- The helper resolves the gate carcass drop-off location and a configured magic campfire feature.
- It asks the existing route finder for a route from gate to campfire and back.
- Added hunter constants for the first five-torch bundle and one-torch return reserve.

## Changed

- `NPC-002` planning now records that route planning exists; the remaining work is the delayed state machine for taking torches, moving, hunting, collecting remains and returning to the gate.
- `npm test` now includes the NPC hunter helper regression script.

## Validation

- Added focused tests for route directions, total travel cost, missing-route results and torch bundle constants.
- TypeScript typecheck covers the new Prisma-backed route-plan helper.
