# 0.13.12 - Gate hunting saturation stand-down - 12026-05-30

This patch gives the gate hunting loop its first "enough for now" state. It is still an ecological pressure loop, not a quest completion flag.

## Added

- Added a conservative saturation helper for the gate hunting loop.
- The helper considers drop-off contribution, nearby mouse/rabbit pressure and nearby depleted-vegetation signals.
- Added saturation-aware inspect text for the gate hunting notice and carcass drop-off feature.
- Added a hunter stand-down path: hunters without claimed carcasses route toward the magic campfire, rest and use a quieter waiting line pool while saturation is active.

## Changed

- Carcass/remains drop-offs are still accepted while saturation is active.
- New supply reward thresholds are suppressed while saturation is active, so the settlement no longer keeps rewarding more hunting when the border has enough pressure for now.
- Player auto mode now avoids choosing the same broad automatic action twice in a row when another candidate action is available.
- Plain `/put`, `put` and `–Ņ–ĺ–ļ–Ľ–į—Ā—ā–ł` now temporarily default to `/put —ā—É—ą–į —Ä—Ė–≤`, so the drop-off hint remains usable even without typed parameters.
- Location features can now use per-feature icons; the torch stand, hunting notice and carcass drop-off near the gate no longer all share the generic landmark glyph.
- `/open` now acknowledges visible non-openable local gates such as the closed settlement gate instead of saying no gate is present.
- `/open`, `open`, `o`, `–≤—Ė–ī–ļ—Ä–ł—ā–ł`, `–≤—Ė–ī—á–ł–Ĺ–ł—ā–ł`, `–≤—Ė–ī–ļ—Ä–ł–Ļ`, `–≤—Ė–ī—á–ł–Ĺ–ł` and related forms now accept optional gate-like targets such as `–≤–ĺ—Ä–ĺ—ā–į`, `–Ī—Ä–į–ľ–į` or `gate`.
- `ECO-003` now records the implemented first slice and remaining threshold/linger tuning.
