# 0.13.15 - Auto stand-up after rest - 12026-05-30

This patch fixes player auto mode after active rest.

## Fixed

- Auto mode now stands a sitting character up before automatic physical actions such as movement or gathering.
- Auto-rested characters no longer repeatedly hit the posture guard after rest completes and leaves them sitting.
