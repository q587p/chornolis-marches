# 0.13.16 - World lexicon case forms - 12026-05-30

This patch updates the older world-lexicon draft for the current codebase and makes it part of the active grammar and seed flow.

## Added

- Added `src/content/lexicon/worldLexicon.ts` as the central source for stable world-facing Ukrainian noun case forms.
- Added `docs/content/world-lexicon.md` with usage rules for world nouns, seed data and gameplay text.

## Changed

- Grammar known-form lookup now reads from the lexicon instead of keeping a separate hardcoded block.
- Seed species definitions now use `creatureSpeciesNameFields(...)` for active animals, current NPC profession species and the forest spirit species.
- README now points to the content/lexicon documentation.

## Validation

- Extended character-name and grammar coverage for lexicon forms.
- Ran the project build and full test suite.
