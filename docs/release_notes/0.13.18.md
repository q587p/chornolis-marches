# 0.13.18 - Hunter claimed-carcass and speech hardening - 12026-05-30

This patch tightens the NPC hunter loop around claimed prey, local field speech and the remaining future structure for actor carry state.

## Changed

- Hunter-claimed carcasses now keep their `claimed_by_hunter:<id>` marker while corpse decay updates the remaining lifetime text.
- Returning hunters can keep seeing their claimed prey until it is deposited or finally disappears, instead of losing it when decay rewrites the corpse action text.
- Direct hunter field lines now use a shared speech helper that records a `SAY` event and increments the creature speech counter shown in ecology/status statistics.

## Documentation

- Updated the gate hunting loop system notes with the 0.13.18 bridge hardening.
- Updated hunter-auto and actor-inventory planning items to keep real claimed-prey carry/ownership state as explicit future work.
