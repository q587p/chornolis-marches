# 0.13.20 - Name review clarity - 12026-05-30

This patch makes character-name onboarding clearer without changing the underlying approval flow.

## Changed

- Prepared character names now explicitly say that scribes have already checked them.
- Custom character names now tell players they may continue immediately while the name still waits for later scribe review.
- The character card uses the same name-review wording, so pending custom names read as pending review instead of a hard error.
- Unknown-input suggestions now attach stable slash commands to more entries, including stat and item-use suggestions.
- The fallback hint now points to `❔ Допомога` (`/help`) and `☰ Меню` (`/menu`) with clearer labels.
- Auto stand-up notices now include the matching `/stand` command hint.

- Regenerated planning exports.
