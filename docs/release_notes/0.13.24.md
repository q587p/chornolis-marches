# 0.13.24 - Hunger cues, tutorial completion and inventory polish - 12026-05-31

This patch makes existing hunger easier to notice without adding a new reminder loop, and gives the tutorial dream one gentle player-facing wellbeing aside.

## Added

- Added threshold-based hunger cues when player hunger rises into moderate or serious states after stamina-spending actions.
- Hunger cue text uses available context:
  - carried edible food points toward `inventory`;
  - carried raw meat plus a local fire points toward cooking;
  - nearby berries or mushrooms point toward local food;
  - otherwise the cue stays atmospheric and does not promise unavailable food.
- Added a one-time tutorial dream wellbeing aside after the first successful tutorial food use from berries, mushrooms or cooked meat.
- Added `/respawn` as a confirmed atmospheric return to the start location for early or weak characters who get lost.
- Added `/tutorialEnd` / `Закінчити навчання` as a confirmed tutorial-completion flow that marks the tutorial complete, returns from the dream when needed, and removes unfinished-tutorial reminders from help/profile surfaces.
- Added `scripts/test/hunger-cues.cjs` and included it in `npm test`.
- Added `scripts/test/beginner-return.cjs` and included it in `npm test`.

## Changed

- Rearranged the character card action keyboard into clearer rows: inventory first, then signals/posture, rest/sleep and auto.
- Moved the scribe technical-details toggle from the character card keyboard into the scribe admin menu.
- Moved `/respawn` near the top of the Telegram side command menu.
- `/tutorialReset` now clears the tutorial wellbeing aside flag with the other tutorial progress events.
- Marked `FOOD-004`, `ONB-003` and `SURV-001` beginner-return slices as testing and regenerated planning exports.
- Bumped package metadata to `0.13.24`.

## Fixed

- Corpse drops no longer get stuck when a carried corpse inventory stack exists but its hidden creature row cannot be matched; the carried item is consumed and a matching corpse resource is placed on the ground.
