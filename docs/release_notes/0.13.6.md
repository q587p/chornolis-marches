# 0.13.6 - Whisper, reply and shout commands - 12026-05-29

Patch release focused on finishing the near-term text command pack for speech.

## Added

- Added `whisper [player] [message]` and Ukrainian `шепнути [персонаж] [текст]` aliases.
- Whispered speech is private to the target player; nearby bystanders only see that a whisper happened, without the message text.
- Added `reply <message>` / `відповісти <текст>` for answering the latest local speech event that addressed the character by name/forms.
- Added `shout <message>` / `гукнути <текст>` / `крикнути <текст>` for region-level speech that spends extra stamina.

## Changed

- `/commands`, input-alias docs and `CMD-001` now list `whisper`, `reply` and `shout` as current commands.
- `docs/planning/next.md` no longer keeps `CMD-001` in the active 0.13 lane now that the compact text-command pack has shipped.
- Inventory fire actions such as lighting/dousing torches and adding twigs now send the action result as its own message, then refresh the inventory view separately.

## Validation

- Parser tests cover English and Ukrainian forms for `whisper`, `reply` and `shout`.
