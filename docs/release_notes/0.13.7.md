# 0.13.7 - Gate hunting loop foundation - 12026-05-30

This patch starts the gate hunting loop as a living-world ecological pressure point, not a formal quest.

## Added

- Added `docs/systems/gate_hunting_loop.md` as the design source for the settlement-gate hunting loop.
- Added planning items for the gate hunting loop, the narrow local-container `put` command, carcass drop-off contribution reactions and future NPC hunter behavior.
- Added an inspectable gate notice explaining herbivore pressure, scarce predators and overgrazing near the settlement edge.
- Added an inspectable physical carcass/remains drop-off feature near the closed settlement gate.
- Added `put` / `/put` parsing for default, numeric and `all` / `все` amounts.
- Added a narrow inventory-to-local-feature transfer path for carried carcasses/remains into the gate drop-off.
- Added contribution storage for carcass/remains drop-offs, recording contributor/source, resource kind and amount.
- Added first and threshold settlement reactions without fixed per-corpse bounty pricing.

## Changed

- `/help`, `/commands` and input-alias docs now include the supported `put` examples for the gate drop-off.
- Planning exports now include the new 0.13.7 gate hunting loop task pack.
- The second tutorial dream location now keeps a narrow reply keyboard with only look, north and south, instead of exposing the full main keyboard too early.

## Validation

- Parser tests cover `put` default amount, numeric amount and `all` / `все` forms.
- Focused helper tests cover valid carcass resource keys and threshold reaction crossing.
- Reply-keyboard tests cover the second tutorial dream location.
