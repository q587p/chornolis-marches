# 0.13.8 - Location route finding foundation - 12026-05-30

This patch adds the first reusable route-finding foundation for future NPC and player auto-mode movement.

## Added

- Added `src/services/routeFinding.ts`, a bounded breadth-first route helper over existing location exits.
- Added a pure graph route finder that returns ordered movement steps or `null` when no route is available.
- Added a Prisma-backed location route lookup for current world exits.
- Hidden exits are excluded by default.
- Caller-blocked directions are excluded by default, and current locked tutorial exits are respected unless explicitly allowed.

## Changed

- `NAV-001` is now in testing as the route-finding foundation for the future NPC hunter loop.
- `docs/planning/next.md` now points the next gate hunting slice toward settlement tuning or NPC hunter behavior built on this helper.

## Validation

- Added focused route-finding tests for simple paths, disconnected destinations, hidden-exit filtering, blocked directions, same-location routes and depth limits.
- `npm test` includes the new route-finding regression script.
