# 0.13.9 - NPC drop-off contribution foundation - 12026-05-30

This patch prepares the gate hunting loop for NPC hunters without turning the loop into a quest or a hidden counter.

## Added

- Added `recordNpcCarcassDropoffContribution()` for future hunter behavior.
- NPC deposits use the same carcass drop-off contribution table and feature key as player `put` actions.
- Added a compact hunter field-line pool for departures, trail choices, returns, deposits and giving up.

## Changed

- NPC contribution records preserve `creatureId` and contributor kind.
- NPC deposits do not grant player inventory rewards.
- `ECO-002` and `NPC-002` planning notes now point to the shared drop-off helper and the existing route-finding foundation.

## Validation

- Extended gate hunting loop helper tests for hunter field-line selection.
- TypeScript typecheck covers the new Prisma-backed helper.
