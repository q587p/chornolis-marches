# 0.14.10 - Hunter Torch Route Cleanup and Dream Action Ladder - 12026-06-01

This release cleans up the first hunter torch route assumption after the starter
camp watchtower landed, and extends the tutorial dream with a short optional
look/examine/sign/trace ladder.

## Added

- Added `HUNTER_TORCH_SOURCE_FEATURE_KEY` so hunter resupply points at the
  watchtower torch source instead of assuming the gate/drop-off location.
- Added an inline `Вгору` movement button for inspectable features with an
  authored upward route, starting with the starter-camp watchtower.
- Added `вверх` as a text alias for `UP` movement.
- Added four tutorial dream locations after the existing safety room:
  `dream_tutorial_attention_path`, `dream_tutorial_signs`,
  `dream_tutorial_traces` and `dream_tutorial_waking_edge`.
- Added inspectable dream features for orientation, signs, traces and the new
  tutorial completion marker.
- Added movement voice lines for the new dream action ladder.
- Added seed tests for the new dream rooms, ladder exits, moved tutorial end
  surface and hunter torch-source markers.

## Changed

- Hunters returning for torches now route toward the starter-camp watchtower,
  resupply there, then route toward the known magic campfire before seeking
  prey.
- The old gate torch stand remains available but is marked as downgraded and no
  longer the hunter resupply source.
- The `Закінчити навчання` surface moved from `Затишок останнього кроку` to
  `Край пробудження`; direct `/wake` and `/tutorialEnd` remain available.
- Updated hunter, fire/light, gate hunting, onboarding and planning docs.

## Risks / Rollback

- Hunters now need a route from their current position to the watchtower source
  before resupplying. Existing route finding should cover this, but live hunter
  movement should be watched after deploy.
- The old gate torch source still exists for player/emergency use; later supply
  work may remove or limit it.
- The tutorial is a few rooms longer, though direct wake/end commands remain
  available. Watch whether new players find the forward completion marker
  clearer or too far away.
- Rollback: revert this patch to restore hunter resupply at the gate and move
  the tutorial completion surface back to `dream_tutorial_safety`. No schema
  migration is included.
