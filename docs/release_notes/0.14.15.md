# 0.14.15 - Ecology start-balance stabilization

Date: 12026-06-01

## Summary

This patch keeps the ecology system shape intact and tunes only the starter balance. Mice and rabbits now begin in authored breeding clusters with enough nearby forage to have a real chance at early reproduction, while foxes and a lone wolf remain present as danger.

It also adds a small compatibility fix for older characters created before the starter knife grant: the first freshening action that reaches the new blade requirement with empty hands can now grant and equip a plain knife, then continue the queued action.

## Changes

- Fixed starter animal seed idempotency so existing starter groups are counted by species, location, alive/corpse state, age and explicit sex.
- Rebalanced starter mice into forest, meadow and riverbank breeding clusters.
- Rebalanced starter rabbits into forest, meadow and riverbank breeding clusters.
- Kept foxes present but away from prey breeding cells, and reduced the initial wolf start to one wolf.
- Added local food-rich resource overrides for the prey breeding clusters without changing global biome defaults.
- Added a legacy freshening knife grant for characters who have no weapon in hand when the new freshening requirement first matters.
- Added `scripts/test/starter-animals.cjs` and included it in `npm test`.
- Updated ecology docs and planning notes for the starter-balance slice.

## Risks

- This is an authoring/balance patch, not a full ecology redesign. Live worlds with already-collapsed populations may still need reset, admin intervention or the future no-breeding-pair restoration fallback.
- Local food-rich pockets can increase prey growth if predators are removed or delayed too much; watch early `mouseBirths`, `rabbitBirths`, predator kills and overgrazing after deploy.
- Predator corpse-feeding remains future work, so hungry predators still primarily pressure live prey unless they already have claimed corpses.
- The compatibility grant only fires when the character has no weapon equipped. If they are holding a non-freshening weapon such as a spear, the action still fails with the normal tool guidance.
