# 0.14.17 - Ecology restoration observability

Date: 12026-06-01

## Summary

This release makes the quiet population-floor safeguard easier to monitor through scribe/admin statistics. It does not add a new player-facing spawn mechanic; it gives maintainers better visibility into which starter animal species still need repeated recovery after ecology pressure.

## Changes

- `/stat` now includes recent and total population-floor restoration counts.
- Protected web `/stat` adds restoration cards, a rate row and a per-species restoration table.
- `/stat.json` now includes `populationRestorationRows` with species key/name, restored count, event count and latest restoration timestamp.
- `populationFloorRestored` is now part of parsed `World Tick` counters.

## Validation

- `node scripts/test/ecology-stats.cjs`

## Risks

- The per-species totals are derived from `Population floor restored` world events, so historical periods before those events were logged cannot be reconstructed.
- This is intentionally admin/scribe observability; ordinary player-facing surfaces should still treat recovery as quiet ecology background.
