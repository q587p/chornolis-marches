# 0.14.2 - Tutorial polish and visible world-time beats - 12026-05-31

This patch keeps the `0.14.x` line small: it makes internal time more visible, smooths several tutorial rooms and records theft/hiding as later planning without pulling it ahead of visibility foundations.

## Added

- Added compact player-facing notices when the internal Chornolis clock crosses dawn, day, dusk or night.
- Added wrong-keyboard-layout command suggestions for common mistyped inputs.
- Added pagination for prepared onboarding names.
- Added future theft/hiding system documentation and planning items.

## Changed

- The dream foraging lesson now refills tutorial berries/herbs at the lesson surface so one player cannot exhaust another player's tutorial resources.
- `Зручна лавка` now only speeds rest and no longer raises max stamina.
- Returning from `Жар легкого перепочинку` back to the bench room preserves stamina instead of replaying the one-third stamina lesson.
- The `Закінчити навчання` button surface moved from the fox-observation lesson to `Затишок останнього кроку`.
- Tutorial voice text now uses blockquote-style presentation where Telegram supports HTML instead of inline quote punctuation for direct Сон/Дрімота lines.
- World-time docs now clarify that chronicle Kyiv grouping is display-only, weather simulation remains deferred and stats reset must not reset the internal world clock.

## Deferred

- No darkness or visibility reduction yet.
- No weather simulation yet.
- No light snapshot helper yet.
- No observation-learning, `/observe` or skill sheet work.
- No theft/hiding runtime behavior yet.

## Risks / Rollback

- Daypart notices are proactive heartbeat messages. They respect AFK/end/onboarding guards, but cadence and wording may still need tuning during a 48-minute Chornolis day.
- Tutorial foraging now self-restores only in `dream_tutorial_foraging`; if that feels too magical later, replace it with per-player lesson inventory or instanced tutorial state.
- Tutorial rest balance changed: the bench no longer grants extra stamina, and returning from deep rest no longer drains stamina. Players must enter westward from the hub to trigger the one-third stamina lesson.
- Tutorial completion now surfaces from `Затишок останнього кроку`; move the button surface again when more final tutorial rooms are added.
- Keyboard-layout suggestions are advisory only and do not auto-run commands.
- Rollback: revert this patch if tutorial pacing or daypart notices need to return to the previous quieter behavior.
