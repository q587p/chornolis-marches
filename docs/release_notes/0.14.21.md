# 0.14.21 - Willow Floodplain first contour

Date: 12026-06-01

## Summary

This release adds the first seed-only Willow Floodplain pocket near the current riverbank and old bridge. It is a narrow map/content slice: new region, new locations, route shape and documentation, without opening the settlement gate or adding new gameplay systems.

## Player-Facing Notes

- A new `Вербові Низи` area is reachable from existing riverbank locations.
- The floodplain adds twelve authored locations with swamp, river and dry-islet texture.
- The new pocket has two returns to known territory and does not bypass the closed settlement gate.

## Technical Notes

- Added the `willow_floodplain` region to split world seed data.
- Added twelve `z = 0` locations and thirty-two exits in `prisma/data/world/`.
- Added `MAP-WILLOW-001` planning coverage and refreshed `docs/world/world_map.md`.
- Kept notification settings, visibility systems, settlement content, observation, fishing, economy, combat and theft out of scope.

## Validation

- Ran `npm run test:seed`.
- Ran `npm run test:seed:types`.
- Ran `node scripts/test/route-finding.cjs`.
- Ran `npm run map:render`.
- Ran `npm run build`.

## Risks

- This branch starts from current `main` at `0.14.19`; the separate `0.14.20` notification-settings PR was still open when the branch began.
- The floodplain currently has no dedicated landmarks, resources, creatures or feature aliases; those are intentionally left to follow-up Willow tasks.
- Coordinates are authored as a compact `z = 0` pocket near existing riverbank anchors, so later map expansion should review how it connects to under-bridge and watchtower routes.
