# 0.14.8 - First-Session Clarity After Darkness - 12026-06-01

This release keeps the first session readable now that darkness can hide
location descriptions, nearby beings, ground objects, resources and tracks.

## Added

- Added one-time first-night guidance for waking-world characters when dim/dark
  visibility is actively reducing what they can see and there is no local light.
- Added one-time tutorial-dream guidance after first `look` and first `examine`
  flows, using Сон and Дрімота voice comments instead of checklist copy.
- Added an inspectable newcomer board at `start_border_camp`.
- Added a beginner torch source at `start_border_camp`. The existing
  closed-gate torch source remains unchanged.
- Added planning items for the completed camp prose pass, the future watchtower
  and torch-source route pass, the next look/examine dream-semantics step and a
  focused first-session smoke checklist.

## Changed

- Reworked the `start_border_camp` description so the start and `/respawn`
  anchor reads as a shared border camp near the bridge, gate, river and forest
  edge.
- Daypart transition notices now skip tutorial dream and future dream-layer
  locations, so waking-world time does not interrupt dream scenes.
- `/help` now checks whether the character is already in the tutorial dream and
  avoids telling them to return there when they are already there.
- Rest-ready and passive recovery messages now keep exact HP/stamina values
  behind technical-detail access and show ordinary players qualitative state
  text.
- Cooking result messages now show a repeat-cooking button when the player still
  has raw meat left.
- `/help` no longer advertises the hidden `/commands` catalog to ordinary
  players.
- Updated onboarding, fire/light, roadmap and next-planning docs to reflect the
  new first-session guidance and deferred camp/dream follow-ups.
- Extended world seed and helper tests around starter-camp feature icons,
  torch-source data, tutorial hint text and first-night visibility rules.

## Deferred

- No watchtower yet.
- No shared supply cache or hidden restock.
- No hunter route rewrite.
- No ordinary sleep.
- No full dream expansion beyond first-use look/examine hints.

## Risks / Rollback

- The first-night guidance is sent as a follow-up to location rendering. It is
  guarded by session presence and a per-character world event, but live cadence
  should be watched after deploy.
- The starter-camp torch source is intentionally temporary and infinite for now.
  Later supply/cache work should turn it into a limited container or bundle.
- This branch is based on the 0.14.7 branch head. If `main` has not yet merged
  0.14.7, merge 0.14.7 first or rebase this branch after that merge.
- Rollback: revert this patch to remove the new guidance, camp copy and starter
  torch source. No schema migration is included.
