# 0.15.1 - Profession line-bank polish

Date: 12026-06-01

## Summary

This small early `0.15.x` patch expands hunter and herbalist line banks so the first profession NPCs feel less repetitive while players map, observe, gather and notice living-world behavior.

## Player-Facing Notes

- Hunters have more short lines for field patrol moments: leaving, reading trails, returning, depositing catches, giving up and standing down.
- Hunter conversation replies now have a wider pool of practical frontier advice.
- Herbalists have more ambient lines about plants, exhausted locations, careful gathering and the borderland's living signs.
- Owl attacks now have small owl-specific wording, so visible nighttime predation reads more like a winged shadow than a generic predator event.
- `Cook all` now accounts for raw meat that is already queued for cooking and for the remaining player action slots before adding more cooking actions.
- Message cadence is unchanged. The patch adds variety, not extra chatter.

## Technical Notes

- Expanded exported hunter line banks in `src/services/carcassDropoff.ts`.
- Expanded exported hunter conversation replies in `src/services/npcHunter.ts`.
- Expanded exported herbalist ambient lines in `src/services/worldTick.ts`.
- Added focused `scripts/test/ambient-lines.cjs` assertions for hunter field banks, hunter replies and the larger herbalist bank.
- Moved `NPC-007` to testing in planning docs and regenerated planning exports.
- Added `OWL-002` as the next owl follow-up for live tuning, daypart checks and small indirect owl signs without expanding into flight, nests or reproduction.
- Added `src/services/predatorActionText.ts` for owl-specific predator action/observer text and focused owl text assertions.
- Added a cook-all planning helper so queued cooking does not overfill the action queue or duplicate already planned raw meat.

## Risks

- This is content variety only. It does not change when NPCs speak, so players may still notice repetition if one situation fires very often.
- The new Ukrainian lines are hand-written and covered for duplicate banks, but live tone should still be watched around hunters and herbalists.
