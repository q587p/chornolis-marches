# 0.15.10 - Camp cat cache presence, mouse priority and pounce

Date: 12026-06-02

This patch keeps the early camp-cat line small: the cat feels more tied to the starter cache and local mice without becoming a pet or a full predator system.

## Summary

- Added an optional camp spirit cat presence line to starter shared-cache inspection when the cat is visibly in the same location.
- Kept cache use unchanged: the cat does not own, guard, reward, punish or block starter-cache supplies.
- Added a first camp-local mouse-priority behavior: if live visible mice are in the cat's current camp/watchtower location, the cat chooses its mouse-listening watch posture before ordinary up/down movement.
- Added a first same-location mouse pounce for the camp spirit cat in the starter camp/watchtower pair, routed through the existing creature attack queue.
- Added cat-specific miss, wound and kill copy, plus contextual full-examination detail based on local mice, daypart, active campfire light and watchtower position.
- Added safe main-bot deep links and `/start cmd_*` routing for public news references to `/say`, `/build_campfire`, `/light_campfire`, `/douse_campfire` and `/dismantle_campfire`.
- Updated camp-cat, beginner-cache, ecology, onboarding and planning docs.

## Risks / Rollback

- The cache cat line appears only when the cat is currently visible in the same location. If the cat is downstairs while the cache is upstairs, the cache text remains unchanged.
- The cat pounce is deliberately local: same-location visible live mice only, inside the starter camp/watchtower pair. It does not chase outside camp, hunt non-mouse prey or become an ordinary predator species.
- The pounce uses the existing attack completion flow, so ordinary miss/wound/kill behavior and queue timing still apply.
- The camp spirit cat remains `SPIRIT`/`SPIRITUAL`; it should stay out of ordinary predator species tables, though its own action counters may still change.
- Rollback is a normal revert of the 0.15.10 commit if the cache copy feels too prominent or the cat's movement/pounce cadence feels off.
