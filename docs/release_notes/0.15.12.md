# 0.15.12 - Camp cat mouse pounce and attentive examination

## Summary

- Added the first same-location mouse pounce for the camp spirit cat in the starter camp/watchtower pair.
- Routed the pounce through the existing creature attack queue and added cat-specific miss, wound and kill copy.
- Added contextual full-examination detail for the camp spirit cat based on local mice, daypart, active campfire light and watchtower position.
- Updated camp cat ecology, onboarding and planning docs for this small behavior slice.

## Risks / Rollback

- This branch is intentionally stacked on the `0.15.11` camp cat cache/mouse-priority PR until that PR merges; rebase onto current `main` after the merge.
- The cat pounce is deliberately local: same-location visible live mice only, inside the starter camp/watchtower pair. It does not chase outside camp, hunt non-mouse prey or become an ordinary predator species.
- The pounce uses the existing attack completion flow, so ordinary miss/wound/kill behavior and queue timing still apply.
- The camp spirit cat remains `SPIRIT`/`SPIRITUAL`; it should stay out of ordinary predator species tables, though its own action counters may still change.
- Rollback is a normal revert of this patch; no migration is required.
