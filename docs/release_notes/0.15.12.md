## Risks

- This is intentionally narrow: only the current raw-meat-to-camp-spirit-cat transfer is canonicalized. General player/NPC gifts, item instance selection, multiple-target ambiguity UI and barter remain future work.
- The parser accepts `–º'—è—Å–æ` as raw meat only inside this `give` path. Inventory eating/cooking semantics still keep their own resource resolution.
- This branch is stacked on the `0.15.11` review branch until that PR merges; rebase onto `main` after `0.15.11` lands.
