## Risks / Rollback

- This is intentionally a tiny bridge, not full barter: only one raw-meat unit to the visible camp spirit cat is supported.
- The action is immediate and records a zero-duration `WAIT` note instead of adding a new Prisma `WorldActionType`; broader queued transfer can be added when exchange expands.
- The target button only appears after inspecting/focusing the cat and only if the player currently has raw meat, so live UX should be watched for discoverability.
- Rollback: revert this patch to remove the give bridge, cat feed affordance, docs/news and version bump. No migration or seed refresh is required.
