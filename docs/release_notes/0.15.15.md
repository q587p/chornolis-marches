# 0.15.15 - Liminal Signs and Quotation Guardrails

Date: 12026-06-02

## Summary

0.15.15 adds a documentation and content-planning layer for liminal signs, quotation use and recurring motifs, and adds a small safety fallback for established characters whose ordinary beginner return is no longer available. It records how owls, dreams, fire, fishing water, maps, tracks and local rumors should carry atmosphere without becoming overt references or constant exposition.

## Changes

- Added `docs/content/signs-dreams-and-quotations.md` as a content/design bible for recurring signs, quotation guardrails and motif reuse.
- Recorded the placement decision that the direct owl refrain belongs at a dream/waking threshold, not in ordinary starter-camp copy.
- Added planning items for the liminal signs layer, dream-threshold owl refrain, conditional owl examine copy, magical campfire examine layers and future deep-water fishing lines.
- Promoted the riverbank fisher NPC and tiny-slice planning path to high priority as a near-term living-profession step.
- Updated README, game design and roadmap docs to point future Codex work toward rare, diegetic and local uses of signs and refrains.
- Added `/call_scribes`, `call scribes` and Ukrainian text aliases as a manual rescue request when ordinary `/respawn` refuses an established character.
- Added Scribe/admin return-help notifications with a return-sign button that cancels active/queued player actions, stops rest/auto and returns the player to the start location when applied.
- Updated established-character `/respawn` refusal text so it explains nearby `/yell` and Scribe return-help options.
- Added carrion-ravine feature buttons for one-carcass and all-carcass contribution, using the existing `/put` carcass/remains path and validating that the feature is still local.
- Added an inspectable starter-camp lunar-circle birchbark feature so players can read the 13 lunar circles in order with atmospheric seasonal work notes.
- Updated Scribe/admin world-time debug output to show the lunar circle ordinal alongside the lunar circle name.
- Added a first-active-campfire ignition feedback layer: when a location gains its first active campfire, the player/admin response now mentions warmth, easier rest and, if local light was absent, improved visibility.
- Reduced stamina costs for player attack, freshening and cooking actions, and expanded the first fatigue warning with contextual recovery buttons for rest, sleep, inventory food and campfire building when available.
- Regenerated planning exports.

## Public News

Added a public `news.md` entry about old signs becoming easier to keep in order, the new player-visible fallback from refused `/respawn` to `/yell` or `/call_scribes`, the carrion-ravine contribution buttons, the starter-camp lunar-circle birchbark, first-active-campfire feedback and softer fatigue guidance.

## Risks

- The liminal motif layer is still mostly planning; it does not yet implement the dream-threshold quote, conditional owl examine text, magical campfire examine copy or fishing lines in runtime gameplay.
- The motif layer can become too referential if overused; the new guardrails intentionally keep direct quotations rare and diegetic.
- Planning item status and priority may need later tuning once the next gameplay slice chooses which motif to implement first.
- Scribe return help is a manual rescue path: it depends on reachable Scribes/admins and should not become ordinary fast travel.
- Applying the Scribe sign cancels the target player's active queue/rest/auto state before moving them to the start location; this is intentional for rescue, but Scribes should use it knowingly.
