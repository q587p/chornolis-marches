# 0.15.25 - Starter Cellar And Map-Maker Landmarks

0.15.25 adds a tiny waking-world verticality slice beside the starter camp.

This is a content/map patch, not a new system. It does not change Spirit Call behavior, follow intent, combat, economy, crafting, caches or tutorial dreams.

## Added

- Added `Погріб прибулих`, a safe `z = -1` cellar below `Межовий табір біля мосту`.
- Added reciprocal `DOWN` / `UP` exits between the starter camp and the cellar.
- Added `Люк до погреба` at the starter camp plus three inspectable cellar landmarks: `Стіна старих зарубок`, `Порожня полиця` and `Обірвана мапа на дошці`.
- Added map-maker and future-follow atmospheric hints without adding follow mechanics or named spirit profiles.

## Changed

- Updated the ASCII world map renderer with a `c` marker for the starter cellar and refreshed `docs/world/world_map.md`.
- Updated world seed validation for the new cellar, exits, features, aliases and no-loot constraint.
- Updated planning/docs, changelog and public news for the 0.15.25 map-content slice.

## Validation

- `node scripts/test/world-seed.mjs`
- `node scripts/world/render-map-ascii.mjs --write`
- `npm run planning:export`
- `npm test`
- `npm run build`
- `git diff --check`

## Risks / Watchpoints

- The starter camp now has both `UP` to the watchtower and `DOWN` to the cellar. Telegram keyboard layout should remain readable with the extra vertical exit.
- The cellar is intentionally safe and lootless. If players expect supplies there, future copy can make the empty-shelf/no-cache boundary even clearer.
- This adds another waking-world `z = -1` location. Future deep-underground or dream-light heuristics should continue to distinguish ordinary lower rooms from reserved dream layers.
