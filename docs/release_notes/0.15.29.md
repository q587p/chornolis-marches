# 0.15.29 — Starter watchtower torch light fix

Date: 12026-06-04

## Summary

- Added a narrow authored local-light exception to the starter watchtower torch stand using reflected glow from the unfading campfire below.
- Kept the fix data-driven through the existing feature-provided light path, so new players can inspect and use the starter torch source at night without changing global darkness rules.
- Preserved the starter cellar and ordinary dark locations as light-gated spaces.

## Validation

- Added `scripts/test/starter-watchtower-light.cjs` to check the watchtower torch source, reflected-light marker, limited starter light scope and cellar darkness clue guards.
- Extended world seed validation so the starter torch stand must remain on the watchtower, provide the narrow light exception and stay a torch source.

## Risks / Watchpoints

- The torch stand now counts as a feature-provided light source for the watchtower location. This is intentional starter infrastructure; avoid copying the same flag to ordinary torch stands unless they truly provide ambient light.
- The cellar remains dark by design. If beginner guidance later sends players into the cellar at night, keep those hints separate from this watchtower-only fix.
