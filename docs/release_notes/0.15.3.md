# 0.15.3 - Camp spirit cat foundation

## Summary

- Added the first camp spirit cat foundation as a visible, inspectable, non-speaking presence in the starter camp.

## Risks / Rollback

- Existing production worlds need a seed/map refresh to receive the new cat in the starter camp.
- This is only the foundation slice: the cat does not yet hunt mice, react to raw meat, accept `give`, flee down or use weather/fire body-language behavior.
- Rollback: revert the seed/species/helper/docs changes if the camp spirit cat should not ship yet.
