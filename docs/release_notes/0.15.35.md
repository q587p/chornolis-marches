# 0.15.35 - First attention-gated location

Date: 12026-06-04

## Summary

This release adds the first tiny MAP-004 runtime proof: a safe waking-world root pocket that is reachable only after a light-aware feature inspection reveals the crawl action. It keeps the change small, non-loot-focused and separate from any broad skill-gate or auto-follow framework.

## Changes

- Added `start_cellar_root_pocket`, a small safe root pocket beneath the starter cellar.
- Added `start_cellar_root_gap`, an inspectable cellar feature that shows only a vague dark outline without enough light/detail visibility.
- Added a `Пролізти` (`/crawl`) action plus English and Ukrainian aliases for carefully entering the revealed gap.
- Kept the entry hidden from ordinary visible exit lists while adding a clear visible return path from inside.
- Added focused tests for seed integrity, dark leak prevention, aliases, no resource nodes and the hidden/visible exit split.

## Risks

- This is an authored proof of MAP-004, not a reusable generic gated-location framework.
- The gate uses the existing location-level light/visibility rules; viewer-specific attention gates remain future work.
- The feature transition updates the player directly after checking posture, sleep, HP and visibility instead of introducing a new queued action type.
- Public news intentionally avoids the exact location key and detailed discovery condition.
