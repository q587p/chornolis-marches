# 0.15.4 — Camp cat boundary and web predator stats

Date: 12026-06-01

## Summary

- Added a web `/stat` predator-by-species table so owl, fox and wolf pressure can be compared in the browser without opening Telegram or raw JSON.
- Added the first behavior boundary for the camp spirit cat: ordinary world ticks keep it inside the starter camp/watchtower pair and correct it back to camp if it somehow leaves.
- Added focused tests for web-stat predator species rendering and camp-cat boundary helpers.

## Validation

- `cmd /c npm run planning:export`
- `node scripts/test/web-stat-render.cjs`
- `node scripts/test/camp-cat.cjs`
- `node scripts/test/ecology-stats.cjs`
- `cmd /c npm test`
- `cmd /c npm run build`

## Risks / Rollback

- The cat movement slice is intentionally small. It does not add mouse hunting, meat interest, `give`, shoo/flee scenes, or companion behavior yet.
- Web `/stat` now exposes another table from existing ecology stats data; if the page feels too busy, this section can be reverted without affecting Telegram `/stat` or `/stat.json`.
- Rollback: revert this patch to remove the cat boundary tick behavior and web table. No migration is required.
