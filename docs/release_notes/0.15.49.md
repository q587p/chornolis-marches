# 0.15.49 - Group and follow polish

Date: 12026-06-06

## Summary

- Reduced follow-assist message noise by using a combined personal success line when an assist queues movement.
- Added throttled follow-assist blocker hints for useful failure cases such as darkness, hidden movement, no stamina, busy actions or unavailable visible exits.
- Improved travel group status so it separates nearby members from members elsewhere.
- Added Telegram group action buttons for pending invites, follow-leader setup, leaving and disbanding, all routed through existing group commands.
- Documented slashless scribe/admin `learning <target>` aliases and slow-command log usage.

## Boundaries

- No group movement, shared combat, shared loot, shared inventory, party chat, hidden-route sharing, public skill sheet, combat effects or new map content were added.
- Follow assist still submits only ordinary movement through existing action rules and never repeats hidden routes.
- Slow logs remain label/duration-only and do not include private player message text.
