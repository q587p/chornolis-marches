# 0.15.5 — Nearby speech ranges

Date: 12026-06-01

## Summary

- Added `/yell` / `yell` / `call` / `гукнути` as a nearby speech range for the current location plus visible adjacent exits, including `UP` and `DOWN`.
- Kept `/shout` region-wide and moved `гукнути` / `покликати` away from region shouting so nearby calls and broad shouts are distinct.
- Added vertical `Гукнути вгору` / `Гукнути вниз` prompt buttons for locations with visible vertical exits, plus speech-range docs and focused parser/helper tests.
- Added future planning items for camp-cat watch posture, beginner-cache presence and starter-camp owl boundary tuning.
- Filtered `SPIRIT` / `SPIRITUAL` beings out of public `/who`, web `/who` and `/who.json`, keeping guardian presences such as the camp spirit cat and Дід лісовик out of the active-character list.

## Risks / Rollback

- Nearby voice uses authored visible outgoing exits only. That is intentionally conservative, but one-way exits may make sound feel asymmetric until a later map/sound audit.
- Vertical prompt buttons ask the player to type the line instead of collecting free text inline; this keeps the Telegram flow simple but is not a full scripted tutorial yet.
- Public `/who` now hides all `SPIRIT` / `SPIRITUAL` creatures. If a future spirit should be publicly listable as an ordinary active character, it will need a dedicated rule instead of relying on the default `/who` query.
- Rollback: revert this patch to remove `/yell`, the nearby speech helper, the vertical prompt buttons and the docs/planning updates. No migration is required.
