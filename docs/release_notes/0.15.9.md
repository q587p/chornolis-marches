# 0.15.9 - Quieter movement and animal speech reactions - 12026-06-02

This patch reduces local Telegram movement spam and adds the first small animal-directed speech reaction layer.

## Summary

- Coalesced visible non-player/non-animal creature movement notifications per location behind a short runtime window.
- Added `NON_PLAYER_MOVEMENT_NOTIFICATION_WINDOW_MS`, defaulting to 60 seconds, for tuning the movement coalescing window.
- Coalesced movement messages now use one compact text block with one tracks button and deliberately omit arrival target buttons in the first slice to avoid stale target buttons.
- Added first animal-directed speech reactions for targeted `SAY` / `WHISPER`: mice/rabbits can flee through ordinary movement, foxes show wary attention, and wolves can warn with a growl.
- Added focused notification-buffer and animal-speech reaction tests.

## Validation

- Ran `node scripts/test/notifications.cjs`.
- Ran `node scripts/test/animal-speech.cjs`.
- Ran `npm run build`.

## Risks / Rollback

- The movement coalescer is an in-memory debounce. If the process restarts during the window, pending flavor lines may be dropped; world state and tracks are unaffected.
- The first coalescing slice omits target buttons from movement messages. Players can still use `/look` / `/examine` to see who is present.
- Animal speech reactions are intentionally small and not taming/conversation. Wolf escalation and richer context remain in `ANIMAL-001` follow-up work.
- Rollback: revert the notification-buffer/action-completion changes and animal speech helper if chat cadence or animal behavior feels worse in live play.
