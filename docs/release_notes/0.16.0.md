# 0.16.0 - NPC mentorship offer foundation

Date: 12026-06-06

## Added

- Added `PlayerMentorship` storage for offered, active, declined and ended player-to-local-character mentorship context.
- Added narrow mentor eligibility based on local character role and `observedActorSkillLevel(...)`:
  - herbalist-like mentors can offer `gathering`;
  - hunter-like mentors can offer `tracking`;
  - ordinary animals and unsuitable or unavailable creatures do not offer mentorship.
- Added mentorship offer buttons and text answer parsing for accept, decline and one clarification after an unclear answer.
- Added `/mentor` / `/mentorship` status and end surfaces, plus Ukrainian text aliases.

## Changed

- Following an eligible local character can now produce a mentorship offer after normal follow intent is set.
- Accepting mentorship stores active context and sets follow intent to the mentor creature.
- Accepting mentorship does not enable follow assist, grant learning progress, create a travel group or share hidden routes.

## Boundaries

- No NPC `TravelGroup` leadership.
- No group movement, forced movement or hidden-route sharing.
- No skill multipliers, learning grants on acceptance, combat effects or public skill UI.
- Future mentorship observation bonuses should build on the stored active context in a separate slice.
