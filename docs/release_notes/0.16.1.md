# 0.16.1 - Mentorship observation bonus

Date: 12026-06-06

## Added

- Added active mentorship lookup helpers:
  - `activeMentorshipForPlayer(...)`;
  - `activeMentorshipForPlayerAndMentor(...)`;
  - `mentorshipObservationContext(...)`;
  - `mentorshipObservationBonusForSource(...)`.
- Added the first active-mentorship observation effect for supported gathering:
  - matching active mentor;
  - matching `gathering` skill;
  - supported herbs, berries or mushrooms source;
  - same canonical `gathering` / `observation` / `resource:*` row with amount `2`.
- Added `Mentorship observation` WorldEvent markers for mentored observation audit.

## Changed

- Pending mentorship answers now run after alias parsing, so known slashless commands are not swallowed as unclear mentorship answers.
- `/mentor` status now includes mentor-follow and follow-assist hints without exposing raw learning numbers.

## Boundaries

- No learning progress is granted just for accepting mentorship.
- No global skill multiplier.
- No tracking mentorship bonus yet.
- No attack teaching, group movement, hidden-route sharing, public skill UI or quest chain.
