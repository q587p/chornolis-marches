# 0.16.12 вАФ Strange totem cap expansion

Date: 12026-06-08

## Summary

- Expanded active Strange Totem caps by region:
  - `dry_luka`: 13
  - `riverbank`: 3
  - `chornolis_border`: 5
  - `willow_floodplain`: 1
- Added regional description pools so ambient totems read differently in dry grass, riverbank mud/reeds, forest-edge thresholds and the rare willow floodplain case.
- Added a small authored riverbank corridor near the old bridge and willow edge so players can walk a longer riverbank route without immediately crossing into dry luka or willow floodplain.
- Added explicit Spirit Call toggle commands `/spirit_on` and `/spirit_off`; bare `/spirit`, `/dukh`, `/poklyk`, `/auto`, `–і—Г—Е` and `–њ–Њ–Ї–ї–Є–Ї –і—Г—Е–∞` now show current status and controls.
- Added an ordinary sleep start hint that shows the approximate world-clock auto-wake time if the player takes no further action.
- Added scribe/admin `/learning_chart` for the shared learning level chart: threshold values, technical labels and qualitative Ukrainian labels.
- Kept dismantle behavior stable: fresh/old twig reward ranges, queued action semantics, seven-day lifetime and old-day shedding remain unchanged.
- Updated the manifest test runner with per-command timing, a slowest-command summary and optional `TEST_JOBS` parallel execution.

## Technical Notes

- `src/services/strangeTotems.ts` now exposes the cap table, regional description pools and a small protected-spawn guard for focused tests.
- `under_bridge_18_05` now counts as `riverbank` for regional behavior, but remains protected by location key for ambient totem spawning.
- Existing Spirit Call aliases remain compatible: `–њ–Њ–Ї–ї–Є–Ї–∞—В–Є –і—Г—Е–∞`, `–і—Г—Е –≤–µ–і–µ`, `/spirit_stop`, `/dukh_stop`, `/auto_stop`, `—Б—В–Њ–њ –∞–≤—В–Њ` and related stop phrases still route to the appropriate start/stop behavior.
- Ordinary sleep still uses the existing 8-hour recovered-enough and 10-hour force wake thresholds; the start reply now surfaces the 10-hour no-action auto-wake clock label.
- `/learning_chart` is deliberately admin-only and text-only; it does not add a public `/skills` sheet or expose raw progress numbers in ordinary player surfaces.
- Ambient totem spawning still attempts at most once per in-game day and still respects active region caps, active players, existing local totems, visible non-animal presences and visible exits.
- Starter camp/watchtower/cellar, old bridge spans, the under-bridge passage, dream tutorial and closed settlement gate locations remain excluded from ambient totem spawning.
- CI keeps the explicit `npx prisma generate` step, runs `npm test` with `TEST_JOBS=4`, then uses `npm run typecheck` instead of a second Prisma-generating build step.

## Validation

- Updated `scripts/test/strange-totems.cjs` for exact caps, static spawn boundaries, protected location/region exclusions, regional description pools and unchanged dismantle reward ranges.
- Updated `scripts/test/fauna-diversity.cjs` so the 0.16.12 cap expansion is the expected strange-totem boundary while fauna starter/restoration assertions stay unchanged.
- Updated `scripts/test/world-seed.mjs` for the extended riverbank route, the under-bridge region boundary and duplicate exit-direction guardrail.
- Regenerated `docs/world/world_map.md` from authored world data.

## Risks / Watchpoints

- More active totems can make wilderness curiosity denser, especially across dry luka and the Chornolis border; watch live `/look` clutter before increasing caps again.
- Parallel test execution could expose hidden test order/global-state assumptions; default local `npm test` remains serial when `TEST_JOBS` is unset.
- This is not a quest system and does not add new rewards beyond existing `twigs` recovery.
- No fauna seed groups, Lisovyk restoration behavior, `/spirit` unlock, mentorship, group movement, combat, economy/profession system, Prisma schema, migrations or WorldEventMarker changes.
