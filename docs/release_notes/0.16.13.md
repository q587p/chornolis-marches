# 0.16.13 — Root-pocket bottle cache foundation

Date: 12026-06-09
Branch: `codex/0.16.13-bottle-cache`

## Summary

- Added `empty_bottle` as the first narrow empty-vessel resource for the future bottles/herbalism lane.
- Added an authored root-pocket bottle niche in `start_cellar_root_pocket`, with distinct inspection text, a take button and `/take_bottle` support.
- Added a temporary source-take carry cap: the non-depleting niche grants bottles only while the player carries fewer than 3 `empty_bottle`.
- Added English/MUD-style and Ukrainian aliases for taking a bottle, plus focused bottle-cache, alias and seed coverage.
- Updated planning, admin/input and location-feature docs so ALC-002 is documented as a small foundation, not a broad crafting or loot system.

## Risks / Rollback

- Existing production worlds need seed/map refresh after deploy for `empty_bottle` and the root-pocket bottle niche to appear.
- The bottle source is not globally stock-limited yet, but the free source will not give more once the player already carries 3 empty bottles. Real stock, dirty bottles, washing and herbalist resupply remain future ALC work.
- This does not add a recipe service, tinctures, `/brew`, shops, barter, economy, public skill UI, Prisma schema or migration.
- Rollback is a normal revert of the release commit; if already deployed, refresh the live seed/map afterward so the authored source and resource mirror the reverted code.
