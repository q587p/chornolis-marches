# 0.16.14 — Minimal resource recipe service

Date: 12026-06-09
Branch: `codex/0.16.14-recipe-001-resource-recipes`

## Summary

- Added `src/services/recipes.ts` as a minimal resource-stack recipe helper.
- Added pure validation helpers for recipe shape, duplicate input normalization, exact missing amounts, actor support and optional feature tags.
- Added a narrow `applyResourceRecipeForPlayer(...)` helper that validates player resources in a Prisma transaction before consuming inputs and producing an output resource stack.
- Added focused recipe tests and included them in the manifest suite.
- Marked RECIPE-001 as testing and updated crafting/planning docs.

## Risks / Rollback

- This is helper-only infrastructure for future ALC work: no `/brew`, `herbal_tincture`, stamina restoration, herbalism success/failure policy, public crafting UI, shops/barter/economy, Prisma schema or migration.
- Output `ResourceType` rows must already exist; this release does not create future tincture resources by itself.
- Future ALC-001 must still use ALC-006 herbalism success/failure policy before consuming ingredients in live gameplay.
- Rollback is a normal revert of the release commit; there is no migration or seed refresh requirement from this helper alone.
