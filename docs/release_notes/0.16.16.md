## Risks / Rollback

- Early brewing can fail; failed validation consumes nothing.
- Ordinary failure keeps the bottle; harsh failure can consume it without adding a broken-bottle resource yet.
- This is only the first stamina tincture, not broad alchemy, healing/night-sight draughts, dirty bottles, washing, water/thirst, NPC brewing, shops/barter/economy, broad recipe UI or public `/skills`.
- Existing production worlds need seed/resource refresh after deploy for `herbal_tincture` to exist as a `ResourceType`.
- There is no Prisma schema or migration change; rollback is a normal revert of the release commit plus any normal seed refresh handling.
