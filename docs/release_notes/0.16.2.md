# 0.16.2 - Tracking mentorship and reply button

Date: 12026-06-07

## Added

- Added the first hunter/tracking mentorship observation bonus:
  - active `tracking` mentorship only;
  - matching mentor creature only;
  - fresh clear ordinary followed movement only;
  - canonical `tracking` / `observation` / `mentorship_followed_movement` progress.
- Added a small `Відповісти` inline button foundation for private directed speech.
- Added pending reply mode helpers:
  - one next free-text message can route through the existing `/reply` path;
  - known aliases and commands run first;
  - mentorship yes/no answers still resolve mentorship offers first;
  - pending reply mode expires quickly and can be cancelled.

## Changed

- Private directed speech can attach a reply button using the existing remembered reply target.
- Mentorship docs now cover hunter/tracking mentorship as observation-based, not passive learning.
- Social docs now describe the reply button as speech UX, not group chat.

## Boundaries

- No learning progress is granted just for accepting mentorship.
- No tracking bonus for hidden, dark, stale or non-mentor movement.
- No attack mentorship, combat effects, group movement, hidden-route sharing, public skill UI or broad quest system.
- No shared chat or party channel.
