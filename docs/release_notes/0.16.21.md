# 0.16.21 - Herbalist brewing hint

## Summary

- Added a rare herbalist-like mentorship hint for players with enough gathering/herbalism context who have not already tried tincture brewing.
- Added player-scoped marker/dedupe behavior so the beginner brewing hint is not repeated.
- Added focused eligibility/text tests and marked ALC-005 as testing.
- Kept this as a hint/onboarding layer only: no NPC brewing route, no new resources, no new recipes and no quest marker.

## Risks / Guardrails

- The hint must remain rare and qualitative.
- It should not expose raw learning thresholds or force a quest flow.
- It should not block independent brewing discovery.
- It does not add NPC brewing routes, herbalist bottle routes, shops/barter/economy, a public recipe UI, a public skill sheet, Prisma schema or migrations.
