# 0.16.32 - Themed NPC weapons and next-lane cleanup

Date: 12026-06-11
Branch: `codex/0.16.32-themed-npc-weapons-and-next-cleanup`
Status: proposed

## Summary

- Seeded existing authored знахар/herbalist and hunter NPCs with narrow thematic tools and weapons.
- Здравомир and Ведана now carry and visibly hold sickles.
- Лукан and Орина now carry hunting spears and knives, and visibly hold hunting spears; Орина keeps her lit torch and spare torch resources.
- The unique-creature seed and reset paths now apply valid `equippedWeaponKey` values only when the key is a known weapon and the creature also carries that resource.
- Existing visible look/examine/target display paths now surface those held tools/weapons through the current weapon helpers.
- Hunter attack observer text remains bounded to equipped hunters: spear wording appears only when `hunting_spear` is equipped, otherwise the existing generic hunter text remains.
- Moved `WPN-003` to `testing`, regenerated planning exports, and cleaned `docs/planning/next.md` after completed `0.16.30`/`0.16.31` work and the no-code performance evidence report.

## Risks And Rollback

- Main risk is seed/display mismatch if an authored NPC equipment key drifts; focused seed tests cover known weapon keys and carried-resource consistency.
- This release does not add combat mechanics, target eligibility changes, miss/damage/armor/wound/round/counterattack rules, PvP, training arenas, weapon skill effects, NPC gear AI, loot drops, weapon swapping, hand-slot implementation, item instances, schema/migrations, scheduler work, action queue/recovery/Telegram/Prisma tuning or SQL/index changes.
- Performance evidence remains no-code.
- Rollback is a normal revert of this bounded seed/display/planning PR.
