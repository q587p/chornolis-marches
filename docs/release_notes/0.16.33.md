## Summary

- Added assorted world-data polish: moved the no-loot empty cellar shelf into the hidden root pocket and kept cellar dry bunks usable for lie/sleep actions in darkness.

## Risks And Rollback

- Main risk is making the cue feel too tutorial-like or too farmable; tests cover rabbit-only eligibility, visibility guardrails, duplicate/cooldown bounds and no raw skill numbers.
- This is one observation pattern only. It does not implement the full following skill, change follow-assist behavior, bypass hidden routes or darkness, add auto-navigation, group movement, combat changes, scheduler/runtime tuning, action queue/recovery/Telegram/Prisma tuning, schema changes or migrations.
- Rollback is a normal revert of this narrow observation-learning PR.
