# 0.16.4 - Mentorship guided practice prompt

Date: 12026-06-07

## Added

- Added the first mentorship-guided practice prompt after a real mentored gathering lesson.
- Added cooldowned `Mentorship practice prompt` markers for player/mentor/skill/context/action prompts.
- Added a `Спробувати зібрати` inline button that submits the existing gather action for supported herbs, berries or mushrooms.

## Changed

- Mentored gathering observation now exposes prompt context only when a lesson marker was actually created.
- Look/examine observation surfaces can send a lesson line followed by the optional practice prompt when the same nearby resource remains available.

## Boundaries

- Prompt creation does not grant learning progress.
- Clicking the prompt uses the existing gather path; success, failure, stamina and ordinary practice learning remain unchanged.
- Tracking prompts, attack mentorship, quests, daily tasks, group movement, hidden-route sharing, economy rewards and public skill UI remain out of scope.
