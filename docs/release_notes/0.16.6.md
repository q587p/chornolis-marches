# 0.16.6 - Resource text lexicon cleanup

## Summary

- Added shared lexicon-backed helpers for resource forms, display names, accusative names and corpse resource names.
- Removed the duplicated corpse/resource display-name map from `corpses.ts` while keeping existing imports compatible.
- Moved currently supported sexed rabbit, fox and wolf grammar forms into `worldLexicon.ts`.
- Added focused regression coverage for resource text and corpse query matching.

## Validation

- `node scripts/test/resource-text.cjs`
- `node scripts/test/target-formatting.cjs`
- `node scripts/test/meat.cjs`
- `npm test`
- `npm run build`
- `git diff --check`

## Risks

- This is a text/lexicon/grammar cleanup only; no gameplay rules, WorldEvent/WorldEventMarker logic, Prisma schema or migrations changed.
- Wider resource/action text cleanup in `locations.ts`, `targets.ts` and `actionCompletions.ts` remains future scoped work.
