# Admin and Debug Commands

Адмінські й debug-команди доступні писарям Порубіжжя. Писарем Порубіжжя вважається гравець із роллю `SCRIBE` у базі або Telegram ID, доданий у `ADMIN_TELEGRAM_IDS`.

Є прихована команда `/adminSet <secret>`: якщо `ADMIN_SET_SECRET` збігається з параметром команди, поточний персонаж отримує роль `Писар Порубіжжя`. Саму команду не треба показувати у `/adminHelp` чи публічному списку команд; секрет зберігається тільки в локальному `.env` і змінних середовища Render.

Публічний Telegram command menu не показує `/adminHelp` усім. Після `/start` або успішного `/adminSet` бот синхронізує scoped command menu для конкретного чату: якщо користувач уже має доступ писаря Порубіжжя, у списку команд з’являється `/adminhelp`. Це lowercase-форма для вимог Telegram command menu; сервер приймає і `/adminHelp`, і `/adminhelp`. Debug-команди `/debugGet` і `/debugSet` не додаються в бокове Telegram-меню: вони лишаються прямими командами, доступні через кнопку в картці персонажа й описані в `/adminHelp`.

## Поточні команди

- `/adminHelp` — список команд.
- `/world` — стан світу й останні події.
- `/stat` — коротка статистика й посилання на веб-/stat.
- `/chat time [hours|all]` — репліки гравців і NPC за часом; старий формат `/chat 1` або `/chat all` лишається сумісним.
- `/chat location [hours|all]` — ті самі репліки, впорядковані за місциною; веб-режим `/chat?mode=location`.
- `/chat character [hours|all]` — ті самі репліки, впорядковані за мовцем/персонажем; веб-режим `/chat?mode=character`.
- `/who` — публічний список персонажів, активних за останню реальну годину; писарі показані окремо, інші персонажі змішані в одному абетковому списку без публічного розділення на гравців і NPC. Telegram-вивід має паґінацію, коли список стає довгим. Веб-версії `/who?page=0` і `/who.json?page=0` доступні всім без `ADMIN_SET_SECRET`.
- `/all` — усі живі персонажі та істоти.
- `/all dead` — усі записи істот, включно з inactive/dead/corpse/gone.
- `/playerAdmin <#id|ім’я|username>` — детальна службова картка гравця з будь-якої місцини; без параметрів показує поточного персонажа писаря. `#25` завжди означає глобальний id персонажа; проста цифра на кшталт `1` або `2` спершу шукає персонажа у поточній місцині писаря за локальним номером, а якщо такого номера поруч немає, працює як старий пошук за id. Картка містить ім’я й відмінки, схвалення імені, стан, точні числа незалежно від локального debug-перемикача, роль, місцина, речі/ресурси, статистику, авто/ручні дії й останні дії. Картка має кнопки `Схвалити ім’я` / `Зняти схвалення`, перемикання авто і підменю `Телепорт` для цього персонажа. Схвалення пише подію в журнал і повідомляє гравця; зняття схвалення спершу просить писаря написати коментар, потім записує його в журнал і надсилає гравцю. Перемикання авто писарем теж пишеться в журнал і повідомляє гравця як увімкнення/вимкнення самостійних дій. Телепорт із картки може перенести персонажа в поточну місцину писаря, у стартову місцину або в місцину, яку писар введе ключем, назвою чи координатами. Telegram `/all` також дає кнопки `👤 #id` для гравців на поточній сторінці.
- `/tutorialReset [#id|ім’я|username]` — скинути прогрес навчального сну собі або вказаному персонажу. Команда прибирає позначку завершення навчання й робить стартову місцину навчального сну базовою для наступного `/sleep tutorial`; якщо персонаж уже перебуває в навчальному сні, його поточна позиція в сні теж повертається на початок.
- `/all` також дає кнопки `🧩 NPC #id` для видимих на сторінці NPC / не-тварин. Їхня службова картка вже показує деталі NPC і зарезервована під майбутні налаштування поведінки, ширші за авто-режим.
- Pending scribe prompts such as name-rejection comments or requested teleport destinations can be cancelled with `/cancel`, `/skasuvaty`, `/vidminyty`, `cancel`, `skasuvaty`, `vidminyty`, `скасувати`, `відмінити`, `відміна`, `стоп` or `не треба`.
- `/teleport [#id|ім’я|username] <locationKey|x,y,z>` — команда писарів Порубіжжя: перенести персонажа до вказаної місцини. Без персонажа переносить поточного персонажа; для назв із пробілами можна писати `/teleport персонаж -> назва місцини`.
- `/debugGet` — показати, чи ввімкнені технічні деталі для вашого персонажа.
- `/debugSet <0|1>` — вимкнути або ввімкнути технічні деталі для вашого персонажа; `true`/`false` теж працюють. Звичайні гравці не можуть увімкнути ці деталі навіть якщо в базі лишився старий прапорець: потрібен доступ писаря Порубіжжя або `ADMIN_TELEGRAM_IDS`.
- Веб-сторінка `/all` показує той самий службовий список на сайті, але перед показом питає `ADMIN_SET_SECRET`; без налаштованого секрету сторінка не відкриває дані.
- Веб-сторінка `/world` показує детальний службовий стан світу, чергу дій і останні події; як і `/all`, вона питає `ADMIN_SET_SECRET`. Публічна головна сторінка `/` показує український статус, версію, спільну кількість активних персонажів у `/who`, посилання, емблему й короткий опис бачення та тону проєкту.
- `/look` або кнопка `👀 Озирнутися` — показати поточну місцину.
- `/examine` або кнопка `🔎 Роздивитися` — уважніше роздивитися поточну місцину.
- `/locationAll` — список усіх місцин і ключів.
- `/addCreature <speciesKey> <locationKey|x,y,z> [count] [YOUNG|ADULT|OLD]` — додати тварин.
  Приклади актуальних місцин: `/addCreature rabbit forest_04_00 3`, `/addCreature mouse 0,0,0 5 YOUNG`, `/addCreature wolf forest_00_08 1 OLD`.
- `/addCreatureHelp` — список speciesKey для тварин.
- `/addCampfire [locationKey|x,y,z|персонаж]` — команда писарів Порубіжжя: додати звичайне вогнище у вказаній або поточній місцині.
- `/addTorch [персонаж]` — команда писарів Порубіжжя: додати факел у `Речі` собі або вказаному персонажу.
- `/addTwigs [персонаж]` — команда писарів Порубіжжя: додати хмиз у `Речі` собі або вказаному персонажу.
- `/forceOld [speciesKey] [count]` — зробити кілька тварин у поточній місцині старими для тесту старіння.
- `/cleanupCreature [speciesKey]` — видалити одну тварину в поточній місцині.
- `/cleanupCreatures` — очистити всіх тварин і нормалізувати унікальних NPC.
- `/reset` — показати підтвердження для скидання NPC, тварин, ресурсів, черг, слідів, подій світу й persistent auto-state до стартового стану.
- `/restAdmin` — одразу відновити снагу до адмінського максимуму.
- `/tick` — вручну запустити world tick і показати короткий звіт останнього тіку.
- `/tickGet` — показати tick-налаштування.
- `/tickSet <ms>` — змінити інтервал tick.
- `/auto` — увімкнути авто-режим гравця.
- `/autoStop` — зупинити авто-режим.
- `/news` — останні новини гри.
- `/restart` — видалити свого персонажа, інвентар і статистику; наступний `/start` почне онбордінґ з нуля.

## Правила для майбутнього

Мінімальний бажаний напрям:

- список Telegram ID адміністраторів у `.env` лишається аварійним/операторським способом доступу;
- база-роль `SCRIBE` є основним способом зробити персонажа писарем Порубіжжя;
- `/adminHelp` показує повний список лише тим, хто має права;
- небезпечні команди на кшталт `/reset`, `/cleanupCreatures`, `/tickSet`, `/addCreature`, `/addCampfire`, `/addTorch`, `/addTwigs` закриті для звичайних гравців;
- звичайні гравці можуть зберегти доступ до безпечних команд на кшталт `/look`, `/me`, `/news`, `/help`.
