# Settlements System

## Goals

Settlements should be safe-ish social anchors, not generic menu hubs.

## Starter Settlement

Planned NPCs:

- herbalist;
- hunter;
- blacksmith;
- innkeeper;
- elder;
- volkhv / ritual figure;
- guards.

## Services

- Rest.
- Barter.
- Rumors.
- Beginner guidance.
- Notice board / public settlement notices.
- Mail / letters.
- Crafting stations.
- Local reputation.

## Future Systems

- Crime response.

- Regional economy.

- Faction presence.
