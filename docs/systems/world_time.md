# World Time System

## Goals

- Add a day/night cycle that affects exploration, safety and atmosphere.
- Make night dangerous and incomplete without light.
- Support time-based events, creatures and resources.

## Planned Rules

- World time advances through ticks.
- Time phases: dawn, day, dusk, night.
- At night, location descriptions are hidden unless there is a light source.
- At night, the player sees only the location name by default.
- Light sources reveal description, visible creatures, resources, exits and tracks.
- Some creatures and events appear only at night.
- Some NPCs and animals may disappear or sleep at night.

## Future Hooks

- Weather modifiers.
- Sacred days / weekday events.
- Moon phases.
- Seasonal events.
